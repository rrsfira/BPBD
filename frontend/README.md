# Siaga Surabaya - Website Sosialisasi
Website Sosialisasi adalah sistem berbasis web yang dirancang untuk mempermudah pengelolaan dan melihat data sosialisasi.

## Installation

Jalankan pada directory

```bash
  npm install
  npm start
```
